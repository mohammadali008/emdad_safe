# EmdadSafe Frontend (RTL + SEO, Next.js App Router)

## Quick Start
```bash
# 1) inside frontend folder
npm install

# 2) set env
copy .env.example .env

# 3) dev
npm run dev

# 4) production
npm run build
npm start
```

## Notes
- SEO ready (metadata, JSON-LD, robots, sitemap)
- RTL via Bootstrap RTL CDN + custom theme
